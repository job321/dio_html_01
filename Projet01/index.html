Uma breve Introdução

A criação do HTML é atribuída a Tim Berners-Lee, o inventor da World Wide Web.

HTML significa: linguagem de marcação de hipertexto, ultilizado para criação de web sites. Seu papel é descrever a estrutura da página atravéz de elementos.

Esses elemetos do html informam ao navegador como exibir o conteúdo. Os elementos do html são formados por todas as informações que estiverem dentro das tags.

As tags são compostas pelos símbolos de: < (menor que) e > (maior que).

Exemplo: <nome da tag> conteúdo da tag </nome da tag >.
(Início)

Os Principais Elementos

(<p>) é usado para parágrafos

Exemplo: (<p>)Aqui se digita o parágrafo.(</p>)

(<h1>),(<h2>),(<h3>),(<h4>),(<h5>) e (<h6>) São usadas para cabeçalhos, títulos e subtíuulos. Respeitando assim o grau de importancia entre elas.

(<a>) A tag de âncora define um link para outra página. O atributo “href” determina o destino desse link.

Exemplo: (<a href="https://web.dio.me">)Site da DIO.me (</a>)

(Início)

Listas

As listas podem ser: Ordenadas e ou não ordenadas.

  • Listas Ordenadas (<ol>)

    • Exibe os elementos enumerados.

  • Listas Não Ordenadas (<ul>)

    • lista não ordenada, mostra os elementos como bulletpoints.

Cada elemento de ambas as listas são separadas pela tag (<li)>
